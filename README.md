# Design, Analysis and Control of Isolated Resonant DC–DC Converters for DC Microgrid Applications

A comprehensive B.Tech major project focused on the design, analysis, modelling, simulation, control, and hardware implementation of isolated resonant DC–DC converters for high-efficiency DC microgrid applications.

---

## Project Overview

This project investigates four isolated full-bridge resonant converter topologies:

- Series Resonant Converter (SRC)
- Parallel Resonant Converter (PRC)
- LLC Resonant Converter
- LCC Resonant Converter

Each topology was designed, analysed, simulated, and compared under identical operating conditions to identify the most suitable converter for a 560 V to 51 V, 1 kW isolated DC power supply.

---

## Project Specifications

 Input Voltage  560 V DC 
 Output Voltage  51 V DC 
 Rated Power  1 kW 
 Isolation  High-Frequency Transformer 
 Primary Switches  SiC MOSFETs 
 Switching Method  Full Bridge 
 Control Method  Frequency Modulation 

---

## Objectives

- Design isolated resonant converters for DC microgrid applications.
- Analyse SRC, PRC, LLC and LCC topologies.
- Perform resonant tank design using First Harmonic Approximation (FHA).
- Develop mathematical and small-signal models.
- Design a closed-loop PI controller.
- Achieve Zero Voltage Switching (ZVS).
- Compare efficiency and operating characteristics.
- Develop a practical hardware architecture.
